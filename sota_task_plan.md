# SOTA Task Plan: CRO Typography and Text Formatting for Conversion
Date: 2026-03-24
Context: Output Marketing website — applying CRO-validated text formatting decisions: what to highlight, enlarge, center, isolate in standalone sections, and how to structure visual hierarchy for maximum engagement and conversion.

## Goal
Establish the state of the art on how typography, text sizing, emphasis, pull quotes, centered text, and standalone text sections affect conversion rates and engagement on marketing websites. Build a complete decision framework usable for any page on the site.

## Key questions
1. What font sizes are proven to maximize readability and trust on marketing sites?
2. Which text elements should be visually emphasized (bold, larger, color) and which should not?
3. When does centered text help conversion vs hurt it?
4. What is the CRO role of pull quotes and large standalone text sections?
5. How should visual hierarchy be structured to move visitors toward CTAs?
6. What patterns of text emphasis do high-converting pages use?
7. What is the proven impact of F-pattern and Z-pattern on where to place key phrases?
