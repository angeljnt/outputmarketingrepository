# Rhetorical Templates

Use this file when the task involves detecting or avoiding phrase-level and template-level LLM fingerprints.

## How to use this file

- Treat each item as a pattern family, not only a fixed phrase.
- When auditing text, flag clusters. One item alone proves little. Several in close range are a strong tell.
- When rewriting, remove the pattern rather than merely replacing it with a near-synonym from the same family.
