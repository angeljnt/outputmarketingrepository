# Structural Fingerprints

Use this file when the problem is not just a phrase but the paragraph shape, pacing, or discourse pattern.

## 1. Paragraph-level fingerprints

### A. Suspense before substance
Pattern:
1. broad opener
2. importance claim
3. reveal sentence
4. actual point only later

Typical example shape:
- broad context
- “here’s the thing” pivot
- “what matters most” claim
- actual argument

Rewrite move:
Start with the actual claim.

### B. Thesis echoing
The same point is stated three times in slightly different wording.

Pattern:
- opening claim
- restatement with “in other words” or “the real point is”
- summary restatement

Rewrite move:
Keep the strongest version once.

### C. Symmetrical binary framing
Pattern:
- not x, but y
- less x, more y
- x matters, but y matters more

Rewrite move:
Make one direct claim instead of performing contrast.

### D. Bullet inflation
A list exists only because the model likes tidy structure.
Each bullet is grammatically parallel but conceptually thin.

Rewrite move:
Collapse weak bullets into one paragraph or keep only the bullets with distinct substance.

### E. Safe balance paragraph
Pattern:
- upside sentence
- downside sentence
- balancing sentence
- mild conclusion

This often sounds generated because it avoids commitment.

Rewrite move:
State the actual judgment and the main tradeoff.

## 2. Sentence-level fingerprints

### A. Uniform cadence
Several sentences have near-identical length and rhythm.
This creates a polished but synthetic hum.

Rewrite move:
Create deliberate variation. Mix short blunt lines with longer analytical ones.

### B. Front-loaded transition chains
Examples:
- however,
- moreover,
- importantly,
- in practice,
- that said,
- ultimately,

Rewrite move:
Delete most transitions. Let the logic carry the movement.

### C. Abstract subject + vague verb
Examples:
- this approach enables
- this framework allows
- this shift highlights
- this dynamic underscores

Rewrite move:
Name the actor and the action.

### D. Noun stack inflation
Examples:
- strategic growth initiative
- scalable customer engagement framework
- long-term value creation strategy

Rewrite move:
Turn nouns back into verbs and decisions.

## 3. Document-level fingerprints

### A. Predictable intro-body-summary arc
The document announces what it will do, does it, then summarizes what it did.
That is not always wrong, but in short business prose it often sounds machine-made.

Rewrite move:
Cut the announcement and often cut the summary.

### B. Every section sounds equally important
LLMs flatten emphasis. Minor points get the same ceremony as major ones.

Rewrite move:
Rank points. Let one point dominate.

### C. Frictionless coherence
Everything connects too neatly. No jumps, no judgment, no selective compression.

Rewrite move:
Allow sharper turns where human thought would naturally pivot.

### D. Generic universality
The prose avoids specifics so it can fit any context.

Rewrite move:
Add concrete examples, constraints, or named tradeoffs.

## 4. Strong cluster signals

A passage is high risk when several of these occur together:
- formulaic opener
- abstract noun + helper verb
- tidy list of three
- balanced caution sentence
- summary line that restates the obvious

## 5. Quick audit checklist

Flag the draft if it has several of these in close range:
- opener before substance
- “insight” sentence with no actual insight
- mirrored contrast syntax
- repeated sentence length
- inflated diction instead of precise diction
- transitions doing too much work
- summary line that says little
- generic positive close
