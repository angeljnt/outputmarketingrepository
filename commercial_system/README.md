# Commercial System

This folder is the product and marketing operating system for Output Marketing.

Its job is to keep the offer, ICP, proof, website, outreach, proposals, and CRM aligned so the agency stops drifting between multiple versions of itself.

## Operating rules

- English is the only operating language for active product and marketing documentation.
- The public offer stays narrower than the full internal delivery capability.
- `30-Day Pilot`, `Foundation`, `Engine`, and `Full Stack` are the core public offer.
- `White-Label / Capacity Track` and `Project Track` are secondary custom offers.
- White-label can be publicly discoverable, but it should not outrank the core B2B SaaS offer.
- Historical proof must be mapped to delivered capabilities first, then to package relevance.
- If a new ICP, service, package, or positioning idea appears elsewhere in the repo, it is not active until it is reconciled into this folder.
